Book_Recommender
================

A simple book recommender built with flask. Model deployed on yhat and application deployed on Heroku.

###To deploy on yhat    
   	export YHAT_USERNAME="YOUR USERNAME"
    export YHAT_APIKEY="YOUR APIKEY" 
    

###To deploy on Heroku
    heroku create
    git push heroku master
